# Deno Hello World Project 🦕

A simple Deno script that logs "Hello, World!" to the console.

## Getting Started

### Prerequisites
- Deno runtime installed. If not installed, run:
  ```bash
  curl -fsSL https://deno.land/install.sh | sh
  ```

### Running the Script

1. Run the script:
   ```bash
   deno task start
   ```

2. For development with auto-reload:
   ```bash
   deno task dev
   ```

## Project Structure

- `main.ts` - Main script file
- `deno.json` - Deno configuration file
- `README.md` - This file

## Available Tasks

- `deno task start` - Run the script
- `deno task dev` - Run the script with file watching for development
