    Class AddOn

    The main entry point for bedrockKit. Represents a Minecraft Bedrock addon consisting of a behavior pack and a resource pack.

    All collections are lazy-loaded on first access and cached for subsequent calls. Texture maps (item_texture.json and terrain_texture.json) are loaded eagerly at construction time since they are needed for cross-referencing.

    Example

    import { AddOn } from "./bedrockKit.ts";

    const addon = new AddOn("./vanilla_behavior_pack", "./vanilla_resource_pack");

    addon.getItem("minecraft:copper_spear")?.getTexturePath();
    addon.getEntity("minecraft:zombie")?.getLootTables();
    addon.getBiome("minecraft:bamboo_jungle")?.getEntities();
    addon.getRecipesUsingTag("minecraft:planks");
