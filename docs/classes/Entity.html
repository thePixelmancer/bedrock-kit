    Class Entity

    Represents an entity that exists across both packs — a behavior (server-side) definition and optionally a resource (client-side) definition.

    Example

    const zombie = addon.getEntity("minecraft:zombie");
    zombie?.getLootTables();        // LootTable[]
    zombie?.getSpawnRule();         // SpawnRule | null
    zombie?.getAnimations();        // [{ shortname, animation }]
    zombie?.getRenderControllers(); // RenderController[]

    ReadonlydocData

    docData: CommentBlock[]

    All JSDoc-style comment blocks found in this asset's source JSON file, as returned by comment-parser. Empty array when no /** … */ blocks are present in the file.

    Files can contain multiple comment blocks — for example one at the top of the file and one before each major component section. All are preserved in document order.
